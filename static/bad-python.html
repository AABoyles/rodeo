Bad news! Rodeo can't start

What's going on?

Unfortunately Rodeo wasn't able to find the python command it needs in order to run. Rodeo uses the same libraries the IPython Notebook does. This means, if you don't have your notebook setup properly, Rodeo probably isn't going to work.

How do I fix it?

The good news is that this happens to a lot of people and it's fairly easy to fix. The root of the problem is that you don't have Python setup correctly (or it's setup in a pretty non-standard way that Rodeo doesn't recognize). You've got a couple options.

(1) The "I just want it to work" option

I recommend installing a pre-packaged version of Python. The following work pretty well:

  • Anaconda
  • Another distro
Once you're done, try re-running Rodeo.
